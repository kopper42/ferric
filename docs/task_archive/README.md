# TASKS Archive

Archived copies of prior `TASKS.md` execution documents for human audit and posterity.

## Naming Convention

Use (per `docs/agentic-workflow-v1.md` §11):

- `tasks_<phase_slug>_<slice_slug>_<YYYY-MM-DD>.md`

Examples:

- `tasks_phase1_project-bootstrap_2026-04-26.md`
- `tasks_phase1_testing-quality-baseline_2026-04-26.md`
- `tasks_phase2_api-baseline_2026-05-01.md`

## Rule

Archive files are historical snapshots only. They are not the live execution source of truth. The active execution document is always repository-root `TASKS.md`.
